# Stoplist

> 하지 말아야 할 행동을 추천하고 달성률을 기반으로 배지를 부여하는 페르소나 기반 목표 관리 서비스

## 🧑‍🤝‍🧑 팀 구성

- 프론트엔드: 2명
- 백엔드: 2명 (본인 – 페르소나 기능 담당)

## 💡 주요 기능

- 페르소나 선택/조회
- 달성률 계산 및 배지 자동 부여
- 주차별 목표 관리
- 헬스 체크 API, 테스트 환경 구성

## 🔧 기술 스택

- Java, Spring Boot, MySQL
- AWS EC2, RDS

## 🌐 배포 환경

- Ubuntu 22.04 EC2 서버 + MySQL RDS
- 배포 자동화 없이 수동 배포 (해커톤 환경)
